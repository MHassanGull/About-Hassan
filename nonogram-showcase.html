04 — Architecture

  
How the System Works

  
4-stage pipeline — from random seed to production-ready PNG + JSON in milliseconds per puzzle.

  

    
01
Grid Generation
A seeded random number generator creates a binary grid with controlled fill density tuned per size. Every puzzle is reproducible from its seed.

    
02
Clue Calculation
Row and column clues are computed via run-length encoding of consecutive filled cells — exactly how real Nonogram clues work.

    
03
Difficulty Scoring
Each puzzle is scored on 4 factors: clue slack, fragmentation, density, and constraint tightness. Auto-classified as Easy / Medium / Hard / Expert.

    
04
PNG + JSON Export
PIL renders a clean, print-ready PNG. JSON metadata is written alongside. SHA-256 deduplication ensures no two identical puzzles per size.
