## What's here

A single-page portfolio with deep-dive pages for each major project, plus an embedded AI chat assistant that visitors can talk to about Hassan's work.

| Page | What it covers |
|------|----------------|
| [`index.html`](index.html) | Hero, about, skills, projects, reviews, contact |
| [`chatbot.html`](chatbot.html) | RAG chatbot architecture (the assistant on this site) |
| [`voice-agent.html`](voice-agent.html) | AI voice ordering agent (Retell AI + FastAPI) |
| [`nonogram-showcase.html`](nonogram-showcase.html) | 2,400+ procedurally generated puzzles |

## Tech

- **Frontend:** Vanilla HTML / CSS / JS, GSAP, anime.js
- **Reviews & contact:** Firebase Auth + Firestore ([`firestore.rules`](firestore.rules))
- **AI chat widget:** Talks to an n8n workflow over HTTPS — the workflow orchestrates Claude + Pinecone (RAG) + Firebase (session storage)
- **Hosting:** GitHub Pages
